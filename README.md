## 📋 Description

Budget Tracker is a personal finance management app that allows users to:
- Add income and expense transactions
- Organize transactions by category (Food, Rent, Transport, etc.)
- Automatically calculate and display the current balance
- Delete transactions

## ⚙️ Installation & Setup

### Prerequisites
- Node.js
- Python 3

### 1. Clone the repository
```bash
git clone https://github.com/saifan340/budget-tracker.git
cd budget-tracker
```

### 2. Start the Backend
```bash
cd backend
python3 -m venv .venv
source .venv/bin/activate
pip install flask flask-cors
python3 app.py
```
Backend runs on: `http://localhost:8000`

### 3. Start the Frontend
Open a new terminal:
```bash
cd frontend
npm install
npm start
```
Frontend runs on: `http://localhost:3000`
